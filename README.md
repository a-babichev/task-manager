# 📝 Task Manager CLI
Простой и удобный консольный менеджер задач, написанный на Python. \
Позволяет управлять списком дел, сохранять прогресс в JSON-файл и взаимодействовать через интуитивно понятное меню.

## ✨ Возможности
* 📋 Просмотр задач: Отображение списка всех задач с их статусом.
* ➕ Добавление: Создание новых задач с автоматическим присвоением ID.
* ✅ Выполнение: Отметка задач как выполненных.
* ❌ Удаление: Удаление задач по уникальному идентификатору.
* 💾 Сохранение: Автоматическое сохранение данных в файл data.json после каждого изменения.
* 🛡️ Валидация: Проверка ввода пользователя (пустые названия, некорректные ID).

## 🚀 Быстрый старт
### Требования
* Python 3.8 или выше
* Никаких внешних зависимостей (используются только стандартные библиотеки)

### Установка и запуск
1. Клонируйте репозиторий или скачайте файлы проекта:
```bash
git clone https://github.com/a-babichev/task-manager.git
```
2. Перейдите в папку проекта.
```bash
cd app/
```
3. Запустите приложение:

```bash
python main.py
```

## 📂 Структура проекта
|Файл|Описание|
|-|-|
|main.py |Точка входа в приложение (запуск цикла меню)|
|interface.py |Пользовательский интерфейс (меню, ввод, вывод) |
|handlers.py |Бизнес-логика (класс TaskManager, управление задачами)|
|models.py |Модели данных (dataclass Task)|
|utils.py |Утилиты (загрузка и сохранение JSON)|
|data.json |Файл базы данных (создается автоматически при первом запуске)|
|README.md  |Документация проекта|

## 🛠 Технологии
* Python 3: Основной язык программирования.
* Dataclasses: Для чистого и структурированного хранения моделей задач.
* JSON: Формат хранения данных для простоты и читаемости.
* Pathlib: Современная работа с путями файловой системы.
* Type Hinting: Аннотации типов для лучшей читаемости кода.
